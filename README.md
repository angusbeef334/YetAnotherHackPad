# YetAnotherHackPad
A macro pad (hackpad) that I made for the Hack Club [Highway](https://highway.hackclub.com) event.

## Features
9 inputs (MX switches)\
0.91" OLED display\
Powered by Seeed Studio XIAO RP2040\
